# Jolly Jumpers

## 구현할 기능
- n개의 정수(n>0)로 이루어진 수열에 대해 서로 인접해 있는 두 수의 차가 1에서 n-1까지의 값을 모두 가지면
그 수열을 유쾌한 점퍼(jolly jumper)라고 부른다.
- 주어진 수열이 jolly jumper 인지 알려주는 프로그램을 만들기
